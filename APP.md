# APP — Treino

## OBJETIVO

Aplicativo pessoal de treino de musculação para Android, instalável como PWA, funcionando offline.

## TELA 1 — INÍCIO

 Mostrar último treino realizado (letra + data)
 Lista de treinos A, B, C, D

## TELA 2 — EXERCÍCIOS

 Última carga e reps

Ao clicar → abrir exercício

Após salvar → voltar para esta tela
